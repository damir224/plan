# Наложение планировок

Веб-инструмент для сравнения 2–3 планировок квартир наложением друг на друга в **реальном масштабе**. Один файл `index.html`, без сборки и зависимостей.

## Возможности
- Сетка в метрах, зум колесом и панорама — всё в реальном масштабе
- Черчение стен по точным размерам: вводишь длину в метрах, орто-режим держит прямые углы
- Несколько квартир разными цветами, полупрозрачные — область пересечения видна насквозь
- Перемещение и поворот квартиры для совмещения планировок
- Авто-расчёт площади и периметра каждой квартиры
- Три планировки по умолчанию («Набережная», «Самоцветы», «Терле») при первом открытии; кнопка «3 планировки» возвращает их
- Экспорт/импорт JSON, автосохранение в браузере

## Запуск
Открыть `index.html` в браузере (двойной клик). Либо поднять локальный сервер:

    python3 -m http.server

и зайти на http://localhost:8000

## Как пользоваться
1. **Рисовать стены** — кликай углы по сетке; длина стены показывается сразу, можно ввести точную в поле сверху и нажать Enter. Замкнуть контур — клик по первой точке или Enter.
2. **Добавить квартиру** — вторую/третью, другим цветом.
3. **Двигать** — перетаскивай и крути выбранную квартиру, чтобы совместить планировки.

> Прототип (по skill «prototype»): минимум полировки, цель — проверить идею наложения планировок.
